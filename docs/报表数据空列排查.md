# 报表「所在部门、学历、薪水」等列为空 排查说明

## 现象

员工信息明细表（1350035590569136128）中，姓名、年龄、性别有值，但所在部门、学历、薪水显示为空。

## 可能原因

1. **控制台 Network Error**：来自 JimuReport 前端或浏览器扩展。扩展（如 Cursor、广告拦截）可能干扰请求。
2. **报表布局**：该报表为每人 3 行（姓名+所在部门 / 年龄+学历 / 性别+薪水），需确认列宽和滚动是否导致右列被裁切。
3. **分页/异步**：首次加载可能只渲染部分数据，后续分页请求若失败会导致部分列空。

## 排查步骤

1. **无痕模式测试**：Chrome 无痕窗口访问 `http://127.0.0.1:8085/jmreport/view/1350035590569136128`，禁用扩展后再看数据是否完整。
2. **检查 Network**：DevTools → Network 查找失败请求（红色），确认失败 URL。
3. **横向滚动**：表格是否有横向滚动条，向右滚动查看所在部门、学历、薪水列。
4. **验证后端**：`curl "http://127.0.0.1:8085/jmreport/show?id=1350035590569136128"` 返回的 `dataList.uiu.list` 中应包含 `department`、`education`、`salary` 字段。

## 说明

- 后端 `/jmreport/show` 已正确返回完整数据（含所在部门、学历、薪水）。
- Ant Design 未安装、Permissions policy 等警告多来自浏览器扩展，一般不影响报表。

员工信息明细表已改为使用本地 mock 数据，**完全离线运行**，不依赖 api.jeecg.com：
