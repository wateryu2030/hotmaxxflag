# 好特卖看板 - 飞书扫码登录配置

为保护敏感经营数据，看板支持「飞书扫码授权登录」：用户点击「飞书扫码登录」后跳转飞书授权页，使用飞书 App 扫码确认即可登录。

## 一、行为说明

- **未配置登录**：不设置 `FEISHU_APP_ID` / `FEISHU_APP_SECRET` 时，看板不启用登录，所有人可直接访问（与之前行为一致）。
- **已配置登录**：设置上述环境变量后，访问首页或数据导入页会先跳转到登录页，用户需通过飞书扫码授权后才能进入看板。

## 二、飞书开放平台配置

1. **创建自建应用**
   - 打开 [飞书开放平台](https://open.feishu.cn/app) → 创建企业自建应用 → 填写名称等。

2. **获取凭证**
   - 在应用内进入「凭证与基础信息」→ 复制 **App ID**、**App Secret**。

3. **安全设置 - 重定向 URL**
   - 在「安全设置」中添加「重定向 URL」。
   - 必须与看板实际访问地址一致，例如：
     - 本机：`http://127.0.0.1:5002/api/auth/feishu_callback`
     - 外网：`https://htma.greatagain.com.cn/api/auth/feishu_callback`
   - **若已填外网地址**，请在 .env 中设置 `HTMA_PUBLIC_URL=https://htma.greatagain.com.cn`，以便后端用同一地址向飞书兑换 code（代理后 request 可能是 http，必须用固定公网 URL）。

4. **权限**
   - 在「权限管理」中为应用开通：**contact:user.base:readonly**（获取用户基础信息），用于登录后显示用户名。

5. **应用可用性（解决「没有 hotmaxxblog 使用权限」）**
   - 若扫码时提示 **「你没有 xxx 的使用权限」** 或 **「当前登录账号为 xxx（某公司），你可选择以下已登录账号进行授权」**，说明当前飞书账号所在企业未被允许使用该应用。
   - 处理方式（应用管理员在飞书开放平台操作）：
     1. 打开 [飞书开放平台](https://open.feishu.cn/app) → 进入应用 **hotmaxxblog**（或你创建的应用）。
     2. 在左侧找到 **「可用性」** 或 **「使用范围」** / **「应用发布」**（不同版本控制台名称可能略有差异）。
     3. 在 **「可使用应用的企业」** 或 **「已启用企业」** 中，**添加需要登录看板的企业**（例如：振鸿科技有限公司）。
     4. 保存后，该企业下的飞书用户即可在扫码时正常授权；或让用户**切换为已授权企业下的飞书账号**再扫码。
   - 若希望仅本企业使用，只需确保创建应用时所在企业已在可用范围内；若需多企业使用（如集团内多家公司），需逐个添加企业。

## 三、环境变量（.env）

在项目根目录 `.env` 中增加（或从 `.env.example` 复制后填写）：

```bash
# 飞书自建应用凭证（必填才启用登录）
FEISHU_APP_ID=你的App_ID
FEISHU_APP_SECRET=你的App_Secret

# 飞书回调与授权链接的站点地址（与飞书控制台「重定向 URL」一致；走 Cloudflare/代理时必填）
HTMA_PUBLIC_URL=https://htma.greatagain.com.cn

# 会话密钥（生产环境务必改为随机长字符串）
FLASK_SECRET_KEY=随机长字符串

# 可选：仅允许指定用户登录，多个 open_id 逗号分隔；不填则任意飞书用户授权即可登录
# HTMA_ALLOWED_FEISHU_OPEN_IDS=ou_xxx,ou_yyy

# 使用 HTTPS 时建议开启（设了 HTMA_PUBLIC_URL 为 https 时会自动开启）
# SESSION_COOKIE_SECURE=1
```

获取用户的 **open_id**：飞书管理后台 → 通讯录 → 用户详情，或通过飞书 API 查询。

## 四、验证

1. 配置完成后重启看板服务。
2. 浏览器访问看板首页（如 `http://127.0.0.1:5002`），应自动跳转到 `/login`。
3. 点击「飞书扫码登录」，跳转飞书授权页，用飞书 App 扫码并确认。
4. 授权成功后应跳回看板首页，右上角显示「欢迎，xxx 退出」。

## 五、企业微信登录（规划）

当前仅实现飞书扫码登录；企业微信扫码登录可后续按同样思路接入企业微信 OAuth，登录页已预留入口「企业微信扫码登录（敬请期待）」。
