# 好特卖看板 - 完整方案（启动 + 外网访问 + 链接说明）

按下面做一次准备，之后每次用「一种启动方式」即可；外网链接在启动后会在终端里显示。

---

## 一、一次性准备（只需做一次）

### 1. 安装 Cloudflare 隧道（用于外网访问）

在终端执行：

```bash
brew install cloudflared
```

### 2. 飞书 / MySQL 等（可选）

- 飞书推送：在项目根目录 `.env` 里配置 `FEISHU_WEBHOOK_URL`（见 `docs/飞书接收报告与看板链接-操作步骤.md`）。
- MySQL：确保本机已安装并启动，且执行过建表（见 README_运行说明.md）。

---

## 二、每次启动看板（任选一种方式）

### 方式 A：仅本机使用（不生成外网链接）

**终端执行：**

```bash
cd /Volumes/ragflow/hotmaxx/hotmaxxflag
bash 启动好特卖看板.command
```

或：

```bash
cd /Volumes/ragflow/hotmaxx/hotmaxxflag
bash scripts/start_htma.sh
```

**访问地址：** 本机浏览器打开 **http://127.0.0.1:5002**

---

### 方式 B：本机 + 外网访问（推荐，会得到外网链接）

**终端执行：**

```bash
cd /Volumes/ragflow/hotmaxx/hotmaxxflag
bash 启动好特卖看板+外网访问.command
```

或：

```bash
cd /Volumes/ragflow/hotmaxx/hotmaxxflag
bash scripts/start_htma_with_tunnel.sh
```

**会发生什么：**

1. 自动释放 5002 端口（避免“Address already in use”）。
2. 启动看板（本机可访问）。
3. 启动 Cloudflare 隧道，**几秒后终端里会出现一行外网地址**，类似：
   ```text
   https://xxxx-xx-xx-xx-xx.trycloudflare.com
   ```

**不要关掉该终端窗口**，保持运行；关掉后外网链接会失效，看板也会停。

---

## 三、访问地址汇总

| 用途       | 地址 |
|------------|------|
| **本机访问** | 固定：**http://127.0.0.1:5002** |
| **外网访问** | 每次启动后**在终端里显示**，格式：**https://xxxx.trycloudflare.com**（每次启动可能不同） |

- **本机**：同一台 Mac 或同一局域网内，用 http://127.0.0.1:5002 或 http://本机IP:5002 即可。
- **外网**：手机 4G、公司电脑等，必须用终端里这次启动给出的 **https://xxx.trycloudflare.com**，复制到浏览器打开。

---

## 四、外网链接具体在哪里

使用「启动好特卖看板+外网访问」或 `start_htma_with_tunnel.sh` 时：

1. 终端里会先出现“正在启动好特卖看板”“正在启动外网隧道”等提示。
2. 几秒后会出现类似：
   ```text
   Your quick Tunnel has been created! Visit it at (it may take some time to be reachable):
   https://zum-processes-aging-space.trycloudflare.com
   ```
3. **上面这一行里的 `https://...trycloudflare.com` 就是本次的外网地址**。  
   - 可选中复制，发到手机或外网电脑浏览器打开。  
   - 本次不关终端，该链接一直有效；关掉终端后下次启动会得到新的链接。

## 五、桌面图标（双击即启动）

1. 在终端执行一次，把快捷方式放到桌面：
   ```bash
   cd /Volumes/ragflow/hotmaxx/hotmaxxflag
   bash scripts/创建桌面快捷方式.sh
   ```
2. 桌面上会出现：
   - **启动好特卖看板**：只开本机，本机访问 http://127.0.0.1:5002
   - **启动好特卖看板+外网访问**：开本机 + 隧道，**双击后看终端里显示的 https://xxx.trycloudflare.com 即为外网链接**

需要外网时：双击「启动好特卖看板+外网访问」，等终端出现外网地址后复制使用即可。

## 六、常见问题

- **“Address already in use” / 5002 被占用**  
  用「启动好特卖看板+外网访问」或「启动好特卖看板」会先自动释放 5002，再启动。

- **双击 .command 后马上退出、没有外网链接**  
  在终端里执行一次：  
  `cd /Volumes/ragflow/hotmaxx/hotmaxxflag && bash 启动好特卖看板+外网访问.command`  
  看是否有报错（例如未安装 cloudflared）。若已安装仍报错，把终端完整输出发出来便于排查。

- **外网链接每次不一样**  
  快速隧道每次启动都会生成新链接，属正常。需要固定域名可参考 `docs/外网访问本地看板.md` 里的“正式 Tunnel + 自定义域名”。

## 七、流程小结

```
一次性：brew install cloudflared；配置 .env（可选）
    ↓
每次需要看板时：
  - 只本机：bash 启动好特卖看板.command  → 打开 http://127.0.0.1:5002
  - 要外网：bash 启动好特卖看板+外网访问.command  → 看终端里的 https://xxx.trycloudflare.com
    ↓
外网地址 = 终端里本次打印的 https://xxxx.trycloudflare.com（复制到手机/外网浏览器即可）
```
