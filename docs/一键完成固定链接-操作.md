# 一键完成固定链接（greatagain.com.cn）

使用 **greatagain.com.cn** 做固定外网地址，无需申请新域名。免费域名 zhenhongtech2026 需在第三方注册（如 Freenom），无法在此自动申请。

---

## 第一步：在浏览器完成 Cloudflare 授权（仅一次）

1. 若终端里已出现 “Waiting for login” 并给出链接，**在浏览器打开该链接**。  
   若没有，在终端执行：
   ```bash
   cloudflared tunnel login
   ```
2. 用 **wateryu2030@gmail.com** 登录（若未登录）。
3. **选择站点 greatagain.com.cn**，点击「授权」。
4. 看到 “You have successfully logged in” 或终端里命令结束即可。

---

## 第二步：运行一键脚本（创建隧道 + 启动看板 + 隧道）

在终端执行：

```bash
cd /Volumes/ragflow/hotmaxx/hotmaxxflag
bash scripts/setup_and_run_cloudflare_tunnel.sh
```

脚本会：释放 5002 端口 → 创建隧道（若尚未创建）→ 写入 config.yml → 启动看板 → 启动隧道。  
若未设置 API Token，会**打印一条 DNS 记录**，需要你到 Cloudflare 手动添加。

---

## 第三步：在 Cloudflare 添加 DNS（若脚本未自动添加）

1. 打开 https://dash.cloudflare.com → 进入站点 **greatagain.com.cn** → **DNS** → **记录**。
2. 点击 **添加记录**：
   - **类型**：CNAME  
   - **名称**：`htma`  
   - **目标**：脚本里打印的 `xxxxxxxx.cfargotunnel.com`（复制粘贴）  
   - **代理**：已代理（橙色云朵）  
3. 保存。

---

## 第四步：访问固定地址

- **本机**：http://127.0.0.1:5002  
- **外网 / 发给同事**：**https://htma.greatagain.com.cn**

把 **https://htma.greatagain.com.cn** 发给同事即可长期使用，无需再换链接。

---

## 可选：自动添加 DNS（免手动）

在 Cloudflare 创建 API Token（权限包含：Zone - DNS - Edit），并查到 **Zone ID**（在站点 greatagain.com.cn 的右侧 “API” 区域）。然后执行：

```bash
export CLOUDFLARE_API_TOKEN="你的API_Token"
export CLOUDFLARE_ZONE_ID="你的Zone_ID"
bash scripts/setup_and_run_cloudflare_tunnel.sh
```

脚本会通过 API 自动添加 CNAME，无需再到控制台点添加。
