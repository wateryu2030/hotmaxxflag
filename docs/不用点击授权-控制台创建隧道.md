# 不用点击「授权」— 在控制台创建隧道（Token 方式）

点「授权」没反应时，可以**完全在网页里创建隧道**，再在本机用一条带 Token 的命令启动，无需浏览器回调。

---

## 第一步：在 Cloudflare 控制台创建隧道

1. 用 **377728157@qq.com** 登录 **https://dash.cloudflare.com**。
2. 在左侧菜单找到 **「网络」**（或 **Networking**）→ **「Cloudflare Tunnel」** 或 **「隧道」**。  
   - 若左侧没有，可尝试：**零信任 / Zero Trust** → 打开 **https://one.dash.cloudflare.com** → 左侧 **「网络」** → **「隧道」**（Connectors → Cloudflare Tunnels）。
3. 点击 **「创建隧道」**（Create a tunnel）。
4. **隧道名称**填：`htma-dashboard`，点下一步。
5. 选择 **Cloudflared** 作为连接器，点下一步。
6. 页面上会显示**安装命令**，类似：
   ```bash
   cloudflared tunnel run --token eyJh...
   ```
   **复制整条命令**（含 `cloudflared tunnel run --token` 和后面一长串），先不要关页面。

---

## 第二步：在控制台里配置「公共主机名」（路由）

仍在隧道配置页面（或该隧道的「配置」页）：

1. 找到 **「公共主机名」** / **「Public Hostname」** / **「路由」** 等，点 **「添加」** 或 **「添加公共主机名」**。
2. 填写：
   - **子域/主机名**：`htma`（或选 `htma.greatagain.com.cn`，依页面字段为准）。
   - **域名**：选 **greatagain.com.cn**。
   - **服务类型**：HTTP。
   - **URL**：`localhost:5002` 或 `127.0.0.1:5002`。
3. 保存。

若页面是「先保存隧道再配置路由」，就先保存隧道，再进入该隧道的配置，添加上述公共主机名。

---

## 第三步：本机先启动看板，再运行复制的命令

1. 打开终端，先启动看板（释放 5002 并监听）：
   ```bash
   cd /Volumes/ragflow/hotmaxx/hotmaxxflag
   bash 启动好特卖看板.command
   ```
   或：
   ```bash
   cd /Volumes/ragflow/hotmaxx/hotmaxxflag
   bash scripts/start_htma.sh
   ```
   保持此终端运行，或让看板在后台运行。

2. **新开一个终端**，粘贴并执行你在**第一步**复制的整条命令，例如：
   ```bash
   cloudflared tunnel run --token eyJh...
   ```
3. 看到隧道已连接后，**不要关这个终端**。

---

## 第四步：访问与分享

- **本机**：http://127.0.0.1:5002  
- **外网/同事**：**https://htma.greatagain.com.cn**

若控制台里「公共主机名」已填 `htma` + `greatagain.com.cn`，且 DNS 里已有 htma 的 CNAME（你之前在 Safari 里加过），一般无需再改 DNS。

---

## 小结

| 步骤 | 操作 |
|------|------|
| 1 | 控制台创建隧道，复制「安装命令」（含 `--token`） |
| 2 | 同一隧道下添加公共主机名：htma / greatagain.com.cn → localhost:5002 |
| 3 | 本机先启动看板，再在新终端执行复制的 `cloudflared tunnel run --token ...` |
| 4 | 用 https://htma.greatagain.com.cn 访问 |

这样全程**不需要**在浏览器里点「授权」，也不会依赖那个无响应的回调。
