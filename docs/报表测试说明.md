# 报表预览测试说明

## 一、重启服务（必须）

修改后必须重启 JimuReport 才能生效：

```bash
cd /Users/document/好特卖超级仓/数据分析/JimuReport/jimureport-example
# 在运行服务的终端按 Ctrl+C 停止
mvn spring-boot:run
```

## 二、无痕模式测试

1. **确保服务已启动**（如上）
2. **运行测试脚本**：

```bash
cd /Users/document/好特卖超级仓/数据分析
npm run jr:test
```

脚本会打开系统 Chrome 无痕窗口，访问报表预览页，并输出检测结果。

## 三、curl 快速验证

```bash
npm run jr:test:curl
```

检查：
- `/undefined` 应返回 200
- view 页面 baseFull 应为 `http://127.0.0.1:8085`

## 四、手动验证

1. 用 Chrome 无痕模式打开：`http://127.0.0.1:8085/jmreport/view/8946110000000000001`
2. 按 F12 打开开发者工具，查看 Console 是否还有 `undefined`、404 等报错
3. 报表应显示「好特卖销售明细」及数据表格
